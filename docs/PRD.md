# SalesMind AI - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
SalesMind AI 是一款专为中文B2B销售团队设计的AI销售助手，通过自动化线索跟进、智能提案生成和会议安排，帮助销售人员提升3倍工作效率。

### 1.2 核心价值
- **自动化跟进**: AI自动执行80%的重复性跟进工作
- **智能提案**: 根据客户需求1分钟生成专业提案
- **数据驱动**: 实时分析跟进效果，优化销售策略

### 1.3 目标用户画像
**主要用户**: B2B销售代表、销售经理、中小企业主
- 年龄: 25-40岁
- 特点: 每天需跟进20-50个线索，疲于应对
- 痛点: 跟进不及时、提案质量参差、错过最佳成交时机

## 2. 功能需求

### 2.1 MVP核心功能

#### 2.1.1 线索管理
- 支持Excel批量导入线索
- 线索标签分类（ hot/warm/cold ）
- 线索详情编辑

#### 2.1.2 AI自动跟进
- 根据线索状态自动发送跟进邮件/消息
- 支持多轮对话，智能回复客户问题
- 预设跟进剧本（新线索/沉默客户/报价后跟进）

#### 2.1.3 智能提案生成
- 根据客户需求自动生成提案文档
- 支持产品库管理
- 一键导出PDF/Word

#### 2.1.4 会议安排
- 智能识别客户时间偏好
- 自动生成会议邀请
- 日历同步（Google/Outlook）

#### 2.1.5 数据看板
- 跟进状态实时看板
- 转化率统计
- AI建议优化点

### 2.2 用户流程

```
注册登录 → 导入线索 → 设置AI剧本 → 启动自动跟进
→ 查看状态 → 人工介入成交 → 生成提案 → 完成交易
```

## 3. 技术架构

### 3.1 系统架构
- **前端**: Next.js 15 + TypeScript + Tailwind + shadcn/ui
- **后端**: FastAPI + Python 3.11
- **数据库**: PostgreSQL (主) + Redis (缓存)
- **AI引擎**: OpenAI GPT-4o + Claude 3.5 Sonnet
- **部署**: Vercel (前端) + Railway (后端)

### 3.2 核心模块
1. **线索引擎** - 线索导入、管理、标签
2. **AI对话引擎** - 上下文管理、多轮对话、剧本执行
3. **提案生成器** - 模板管理、动态生成、文档导出
4. **日程协调器** - 时间解析、冲突检测、邀请发送
5. **分析引擎** - 数据统计、效果追踪、AI优化建议

## 4. 界面设计

### 4.1 页面结构
- **Dashboard** - 数据概览、待办事项、快速操作
- **Leads** - 线索列表、筛选、批量操作
- **AI Playbooks** - 跟进剧本配置、触发条件设置
- **Proposals** - 提案模板、生成历史
- **Analytics** - 转化漏斗、团队业绩

### 4.2 设计原则
- 简洁专业，突出销售效率
- 移动端适配（销售常外出）
- 深色模式支持
